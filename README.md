# Taimoor

A personal AI agent that helps Taimoor Abdullah — M.Ed student in
International Education Policy and Management at Peabody College,
Vanderbilt, and part-time contributor at Nashville Peer and Taleemabad —
handle the academic and professional workload around his actual work:
reading and summarizing assigned material, tracking tasks and deadlines,
taking notes in professional meetings, synthesizing documents, running data
analysis, and keeping a public professional profile (LinkedIn, published
writing) current and accurate.

See [`planning.md`](./planning.md) for the detailed implementation plan, and
[`CLAUDE.md`](./CLAUDE.md) for the career-strategy context and rules this
project operates under.
